Arquitetura do Sistema

Este projeto propõe um sistema de gestão imobiliária no modelo SaaS com integração de Inteligência Artificial.

A arquitetura do sistema segue o modelo de três camadas:

Frontend
Interface do usuário construída com Streamlit.

Backend
API desenvolvida em Python utilizando FastAPI.

Banco de Dados
PostgreSQL responsável pelo armazenamento das informações da aplicação.

O sistema será multi-tenant, permitindo que múltiplas imobiliárias utilizem a mesma plataforma com isolamento de dados.
